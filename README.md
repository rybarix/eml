# EML

EML is attempt to demystify Tokenizer, Parser, Bytecode and Stack-based VMs by building minimal, easily, extendable "language".

I chose [Odin](https://odin-lang.org/docs/install/) language for implementation due to its simplicity. But you can easily rewrite the code to any other language and follow along.

1. install odin [here](https://odin-lang.org/docs/install/)
2. `git clone https://github.com/rybarix/eml`
3. `odin test main.odin -file` to run tests
